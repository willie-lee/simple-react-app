## Installation

1. Install node modules <br />
    ```yarn install```
2. Run the project <br />
    ```yarn start```
3. For running unit and integration testing <br />
    ```yarn test```
4. For running e2e testing <br/>
    ```yarn add cypress --dev``` <br />
    ```yarn test:e2e```
